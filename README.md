# Context-Aware-Tourism-Recommender
A Python replication of a Context-Aware Tourism Recommender System (Paper 28), using Deep Learning (PyTorch) and the Yelp dataset to balance geographic constraints with user preferences.
Dataset used: Yelp Dataset
A trove of reviews, businesses, users, tips, and check-in data!
Dataset link: https://www.kaggle.com/datasets/yelp-dataset/yelp-dataset
